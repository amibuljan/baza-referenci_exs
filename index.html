Postavljanje baze referenci

Unesite podatke za spajanje na Supabase bazu podataka. Ovo je potrebno samo jednom.

1. Supabase URL Pronađi na: Supabase → Settings → API → Project URL
2. Supabase Anon Key Pronađi na: Supabase → Settings → API → anon public key
3. Naziv firme Prikazuje se u headeru aplikacije

Podaci se čuvaju lokalno u vašem browseru
